Hola GIT !

Para abrir una carpeta desde GIT Bash con CODE VISUAL ES (Code .)

Comandos Basicos
GIT init                        CREAMOS UN REPOSITORIO
GIT ADD                         AGREGAMOS UN ARCHIVO AL AREA DE SEGUIMIENTO
GIT STATUS                      INFO DEL REPOSITORIO 
GIT COMMIT o GIT COMMIT -m      mensaje representativo
GIT PUSH origin master          ingresarlo los archivos al remoto
GIT PULL/CLONE origin master          trae los commit nuevos del remoto
GIT RESET  <file> o <commit>    SACAR ARCHIVO DEL GIT  
GIT CHECKOUT                    TE LLEVA UN UN PUNTO DE LOS COMMIT
